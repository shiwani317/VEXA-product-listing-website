# VEXA-product-listing-website


VEXA is a modern, luxury-themed product catalogue web application built using pure HTML, CSS, and JavaScript. It delivers a smooth and interactive shopping experience with dynamic filtering, sorting, and a clean premium UI.

---

## ✨ Features

- 🛍️ Dynamic product listing
- 🔍 Category-based filtering
- 💰 Price range filtering (with slider)
- 📊 Sorting (price, name, rating)
- ⚡ Fast client-side query handling (simulated backend)
- 🧠 Efficient filtering using category indexing
- 🛒 Add to cart functionality
- 🎨 Premium luxury UI design
- ⏳ Skeleton loading for better UX
- 🔔 Toast notifications
- 📱 Fully responsive layout

---

## 🚀 Tech Stack

- HTML5
- CSS3 (Custom design system)
- JavaScript (Vanilla JS)

---

## 🧠 How It Works

Instead of filtering data directly in the UI, this project simulates backend behavior using an optimized query function. It uses:

- Category indexing for faster lookup
- Debounced inputs for performance
- Dynamic DOM updates without page reload

---

## 🎯 Project Goal

To build a high-quality frontend application that mimics real-world e-commerce behavior while maintaining performance and scalability principles.
